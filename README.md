# 🎓 Student Management System  

A simple **Flask-based web application** for managing students and teachers.  
This project allows you to add, view, update, and delete student and teacher records easily with an intuitive web interface.  

## 📌 Features  
- 👨‍🎓 Manage **Students** (Add, View, Update, Delete)  
- 👩‍🏫 Manage **Teachers** (Add, View, Update, Delete)  
- 🔐 **Authentication system** (Sign In / Sign Out)  
- 🗄️ SQLite database integration  
- 🌐 Simple and clean user interface  

## 🚀 Tech Stack  
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS (Bootstrap for styling)  
- **Database:** SQLite3  
- **Version Control:** Git & GitHub  

## ⚡ Setup Instructions  

1. **Clone the repository**  
   ```bash
   git clone https://github.com/abortedzesus/Student-Management-System.git
   cd Student-Management-System
2. **Create a virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate   # For Linux/Mac
   venv\Scripts\activate      # For Windows

4. **Install Dependencies**
   ```bash
   pip install -r requirements.txt

6. **Run the server**
   ```bash
   python server.py
